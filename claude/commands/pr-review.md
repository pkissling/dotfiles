# PR Review

Enter plan mode first.

I am about to start review of your PR. I will give you the remarks as we go.
You stay in plan mode and patiently wait until i gave you all the remarks. You keep asking for remarks until I'm done.
For every remarkt that I give, you ask clarifying questions until the goal is then. Then you create a dedicated task for the remark.
When i tell you that I'm done with the review, you go ahead and implement them. Once implemented, you run all quality gates and once they pass you tell me that you are ready.
This means that I would then review the uncommited changes. Once I approve, you add all changes with a new commit and hence update the PR accordingly.
